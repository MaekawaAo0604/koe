# STUDIO で Koe を埋め込む

STUDIOの「埋め込み（Embed）ボックス」を使って、お客様の声をサイトに表示します。

---

## 手順

### 1. STUDIOエディタでページを開く

お客様の声を表示したいページをSTUDIOのエディタで開きます。

### 2. 埋め込みボックスを追加

1. 左パネルの「追加」（+ボタン）をクリック
2. 「ボックス」→「埋め込み（Embed）」を選択
3. お客様の声を表示したい位置にドラッグ＆ドロップ

### 3. Koeの埋め込みコードを貼る

1. 埋め込みボックスを選択した状態で、右パネルの「埋め込み」をクリック
2. コードエディタが開くので、Koeのダッシュボードでコピーした埋め込みコードを貼り付け:

```html
<script src="https://あなたのURL/widget.js" data-project="あなたのプロジェクトID" data-widget="あなたのウィジェットID"></script>
```

3. コードエディタの外側をクリックして確定

### 4. プレビューで確認

エディタ上部の「プレビュー」ボタンで表示を確認します。

### 5. 公開

問題なければ「公開」ボタンで本番サイトに反映します。

---

## うまく表示されない場合

### Embed のタイプを確認

STUDIOの埋め込みボックスは内部で `sandbox` 属性付きの iframe として動作する場合があります。

**対処法**:
- 埋め込みボックスの設定で「タイプ」が選べる場合、JavaScript の実行が許可されるタイプを選択してください
- 高さが `0` になっている場合は、右パネルで明示的に高さを指定してください（例: `400px`）

### 高さ・余白の調整

- 埋め込みボックスの**高さ**: お客様の声の件数に合わせて十分な高さを設定してください
  - 目安: 3件表示なら `500px`、5件以上なら `800px` 程度
- 上下の**余白（マージン/パディング）**: 必要に応じて右パネルで調整

### 表示が空白になる

- ブラウザのアドレスバーで直接ページURLを開いて確認してください（STUDIOのエディタ内プレビューでは表示されない場合があります）
- それでも表示されない場合は [トラブルシューティング](troubleshooting.md) を参照

---

## 免責事項

STUDIOでは外部コード（JavaScript）の動作がサポート対象外となる場合があります。埋め込みの表示・動作はSTUDIOのアップデートにより変わる可能性があるため、公開後に定期的に表示を確認することをおすすめします。
