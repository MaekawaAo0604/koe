# Koe - Business Roadmap

## Phase 0: 開発 (Week 1)

### 目標
- MVP完成、デプロイ

### 作るもの
- [ ] 収集フォーム（カスタマイズ可能な質問項目、★評価、テキスト入力）
- [ ] 管理画面（一覧・承認/非承認・タグ付け）
- [ ] 埋め込みウィジェットJS（1行コピペでサイトに表示）
- [ ] Wall of Love ページ生成
- [ ] Stripe連携（課金）
- [ ] 「Powered by Koe」バッジ（無料プラン）

### 作らないもの（後回し）
- 動画テスティモニアル
- SNSからの自動インポート
- AI分析
- アフィリエイト機能

### 最重要ポイント
- オンボーディングに全力: 「登録 → 5分以内にウィジェットをサイトに貼れる」
- Senjaの失敗教訓: 最初の6ヶ月MRR $0だった原因はオンボーディングの放置

## Phase 2: PLGエンジン始動 (Month 2-3)

### 目標
- MRR ¥100,000
- 無料ユーザー 400人

### PLG（Product-Led Growth）戦略
```
無料ユーザーがウィジェットを埋め込み
  → サイト訪問者が「Powered by Koe」を見る
  → クリック → LP → 新規登録（無料）
  → そのユーザーもウィジェットを埋め込み
  → バッジがさらに拡散...（無限ループ）
```

### やること
- [ ] オンボーディング改善（数値計測→改善のサイクル）
- [ ] 週1でXに進捗投稿（MRR、ユーザー数、機能更新）
- [ ] 登録→ウィジェット埋め込みまでのファネル分析
- [ ] 離脱ポイントの特定と改善

### ターゲット別アプローチ
| ターゲット | チャネル | メッセージ |
|-----------|---------|-----------|
| 個人開発者 | X, Zenn, Qiita | 「LPの成約率上げるなら、お客様の声を5分で埋め込み」 |
| フリーランス | X, ココナラ | 「ポートフォリオに実績の声を自動収集→表示」 |
| SaaS企業 | X, BOXIL | 「LPのコンバージョン改善」 |
| コーチ/コンサル | X, note | 「クライアント成功事例を自動収集」 |
